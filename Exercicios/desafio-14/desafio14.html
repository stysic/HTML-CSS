vamos ser mais claros: dentro de uma table vem as tr - table rows ( linhas de tabela ) elas são as horizontáis, e nelas há dados.

UMA TR (LINHA DA TABELA HORIZONTAL) PODE HAVER VÁRIOS DADOS, E ELA CRESCE CONFORME VOCÊ VAI ADICIONANDO AS TH/TD

E para crescer de forma Vertical, é adicionado as tr uma em baixo da outra

quanto mais TR mais linhas verticais são adicionadas
